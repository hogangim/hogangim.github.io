# 개인정보처리방침

**오답문제 인쇄기** (이하 "앱")는 사용자의 개인정보를 중요하게 생각하며, 개인정보보호법을 준수하고 있습니다.

## 1. 수집하는 개인정보

### 1.1 자동 수집 정보
- **기기 정보**: 기기 모델, 운영체제 버전, 앱 버전
- **사용 통계**: 앱 사용 빈도, 기능 사용 패턴 (개인 식별 불가능한 형태)

### 1.2 사용자 제공 정보
- **사진 데이터**: 사용자가 촬영한 문제 이미지
- **저장된 데이터**: 앱 내에서 생성된 레이아웃 설정 및 인쇄 기록

## 2. 개인정보 수집 목적

### 2.1 서비스 제공
- 문제 이미지 촬영 및 저장
- A4 레이아웃 생성 및 미리보기
- PDF 생성 및 인쇄 기능 제공

### 2.2 서비스 개선
- 앱 성능 최적화
- 사용자 경험 개선
- 오류 및 버그 수정

## 3. 개인정보 보관 및 이용 기간

### 3.1 로컬 저장
- **사진 데이터**: 사용자가 직접 삭제할 때까지 기기에 저장
- **앱 설정**: 앱 삭제 시까지 기기에 저장

### 3.2 서버 저장
- 본 앱은 사용자 데이터를 외부 서버에 전송하거나 저장하지 않습니다
- 모든 데이터는 사용자 기기에만 저장됩니다

## 4. 개인정보 제3자 제공

본 앱은 사용자의 개인정보를 제3자에게 제공하지 않습니다.

### 4.1 예외사항
다음의 경우에만 개인정보를 제공할 수 있습니다:
- 사용자의 사전 동의가 있는 경우
- 법령의 규정에 의거하거나, 수사 목적으로 법령에 정해진 절차와 방법에 따라 수사기관의 요구가 있는 경우

## 5. 권한 사용 목적

### 5.1 카메라 권한
- **목적**: 문제 이미지 촬영
- **사용 범위**: 앱 내 카메라 기능에서만 사용
- **저장**: 촬영된 이미지는 사용자 기기의 갤러리에 저장

### 5.2 저장소 권한
- **목적**: 촬영한 이미지 저장 및 PDF 파일 생성
- **사용 범위**: 앱에서 생성한 파일에 대해서만 접근
- **보안**: 다른 앱의 파일에는 접근하지 않음

## 6. 개인정보 보호를 위한 기술적 조치

### 6.1 데이터 암호화
- 앱 내 저장되는 설정 데이터는 암호화되어 저장됩니다
- 이미지 파일은 기기의 보안 정책에 따라 보호됩니다

### 6.2 접근 제한
- 앱은 필요한 최소한의 권한만 요청합니다
- 사용자가 권한을 거부해도 기본 기능은 사용 가능합니다

## 7. 사용자 권리

### 7.1 개인정보 열람 및 삭제
- 사용자는 언제든지 앱 내에서 저장된 이미지를 확인하고 삭제할 수 있습니다
- 앱 삭제 시 모든 개인정보가 함께 삭제됩니다

### 7.2 권한 철회
- 사용자는 기기 설정에서 언제든지 앱 권한을 철회할 수 있습니다
- 권한 철회 시 관련 기능이 제한될 수 있습니다

## 8. 개인정보 처리방침 변경

### 8.1 변경 통지
- 개인정보 처리방침이 변경되는 경우 앱 업데이트를 통해 공지합니다
- 중요한 변경사항은 별도 알림을 통해 사전 고지합니다

### 8.2 동의 철회
- 변경된 정책에 동의하지 않는 경우 앱 사용을 중단하고 삭제할 수 있습니다
